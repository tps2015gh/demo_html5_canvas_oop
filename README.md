# demo_html5_canvas_oop

## Project Overview
This repository demonstrates an HTML5 Canvas animation using object-oriented programming (OOP) principles in JavaScript. It features:
-   Accepts mouse move events for interaction.
-   Checks mouse position against rectangle locations.
-   Utilizes animation (now with `requestAnimationFrame` for smoother rendering).
-   Employs OOP with JavaScript prototypes.
-   Includes static functions within objects.
-   Demonstrates functional programming concepts.
-   Originally did NOT include Drag and Drop functionality (an area for potential extension).
-   Features boxes moving by random initial directions and speeds, now with boundary checks.

## Project Summary / Resume
This project began as a foundational demonstration of HTML5 Canvas capabilities combined with Object-Oriented Programming in JavaScript. It effectively illustrates dynamic visual elements, event handling, and structured code organization. The recent updates, guided by AI testing and programming, have refined the animation loop for consistency across various environments, corrected variable scoping issues, and introduced boundary collision detection for a more robust visual experience. This application serves as an excellent starting point for interactive web graphics and game development, showcasing how a well-structured OOP approach can manage complex on-screen elements and their behaviors.
